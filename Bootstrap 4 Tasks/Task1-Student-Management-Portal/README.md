# Student Management Portal

A responsive Student Management Portal built using HTML, CSS and Bootstrap 5.

## Pages
- index.html - Home page (navbar, hero, features, footer)
- registration.html - Student registration form with validation
- dashboard.html - Dashboard with statistics cards and student table (with search)
- profile.html - Student profile card
- contact.html - Contact form

## Bootstrap Components Used
Navbar, Dropdown, Grid System, Typography, Buttons, Cards, Forms,
Form Validation, Tables, Badges, Alerts.

## How to Run
Open index.html in any web browser.

## Technologies Used
HTML5, CSS3, Bootstrap 5
