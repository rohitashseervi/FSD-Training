# Online Course Registration Portal

A responsive course registration website for a training institute, built with HTML, CSS and Bootstrap 5.

## Pages
- index.html - Home (hero, featured courses, testimonials, footer)
- courses.html - All available courses
- register.html - Registration form with success alert
- about.html - History, Vision, Mission, Faculty
- contact.html - Contact form + map placeholder

## Bootstrap Components Used
Navbar, Cards, Forms, Buttons, Alerts, Grid System.

## How to Run
Open index.html in any web browser.

## Technologies Used
HTML5, CSS3, Bootstrap 5
