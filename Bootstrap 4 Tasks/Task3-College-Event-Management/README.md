# College Event Management System

A responsive website to manage college events, built with HTML, CSS and Bootstrap 5.

## Pages
- index.html - Home (carousel banner, hero, event highlights)
- events.html - All events as cards
- register.html - Event registration form with success message
- schedule.html - Event schedule table (striped, hover)
- gallery.html - Photo gallery
- faq.html - FAQ section using Accordion
- contact.html - Contact form + coordinator details

## Bootstrap Components Used
Navbar, Carousel, Cards, Forms, Tables, Accordion, Alerts, Buttons, Grid System.

## How to Run
Open index.html in any web browser.

## Technologies Used
HTML5, CSS3, Bootstrap 5
