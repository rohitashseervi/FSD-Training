# Personal Portfolio Website

A responsive single-page portfolio website built with HTML, CSS and Bootstrap 5.

## Sections
- Home - Profile photo, name, designation, resume download
- About - Education, career objective, interests
- Skills - Progress bars (HTML, CSS, Bootstrap, JavaScript)
- Projects - 4 project cards with GitHub links
- Achievements - Certifications, workshops, internships, awards
- Contact - Contact form with validation

## Bootstrap Components Used
Navbar, Cards, Progress Bars, Forms, Buttons, Grid System.

## How to Run
Open index.html in any web browser.

## Technologies Used
HTML5, CSS3, Bootstrap 5
